файл gitignore - это текстовый файл,который указывает GIT, какие файлы или папки стоит игнорировать в проекте; git remote rm origin - удаление подключения к удаленному репозиторию; MERGE CONFLICT происходит при автоматическом слиянии веток и разрешается в ручном режиме; отличия между ветками - git diff master ..name; удаление локальной ветки - git branch -d name1; удаленной - git branch -D name2; untracked - git rm --cached file.txt.
